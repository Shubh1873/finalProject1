    Error Handle.....................

  • If you found - (8001 port already running) then type this command
    •             
              $ reboot -f

  • To start app in the production
    •             
              $ pm2 start index.js

  • To stop app in the production
    •             
              $ pm2 stop index.js

  • To automatically restart after a fix time
    •           This is for 5 minute interval - every 5 minute it will restart your app for smooth search.
            
              $ pm2 restart index.js --cron "*/5 * * * *"

  • To monitor your app
    •               
                $ pm2 monit

  • To restart your reverse proxy
    •             
              $ sudo service nginx restart
